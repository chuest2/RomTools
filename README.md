## Introduction 👋

适用于 Xiaomi 14 ，机型代号 houji 。

在 termux 中 clone 本项目，先执行 setup.sh 安装环境，再执行 start.sh <包名> 即可。

## Changes

- 去除安卓 14 开机签名校验
- 精简部分无用 APP
- 关闭内存扩展
- 可选 KernelSU 或 Apatch 获取 root
